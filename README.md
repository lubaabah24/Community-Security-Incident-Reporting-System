# Community Security Incident Reporting System

## Project Overview
The Community Security Incident Reporting System is a web-based application developed to enable residents of Ojo Local Government Area to report security incidents online. The system allows users to register, log in, submit incident reports, track the status of reports, and provide feedback. Administrators can manage submitted reports.

## Features
- User Registration and Login
- Secure Authentication
- Incident Reporting
- Evidence Upload
- Track Report Status
- User Dashboard
- Feedback System
- Admin Panel
- MySQL Database Integration

## Installation
1. Install XAMPP.
2. Copy the project folder to `C:\xampp\htdocs\`.
3. Import `database/security.sql` into phpMyAdmin.
4. Start Apache and MySQL.
5. Open:
   http://localhost/Community-Security-Incident-Reporting-System/
